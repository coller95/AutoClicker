## Installation
1. Download `AutoClicker.exe`
2. Run the executable - that's it!

No installation, no admin rights required (unless automating privileged applications).

## Quick Start
1. Launch AutoClicker.exe
2. Press F1 to start recording
3. Perform your clicks and key presses
4. Press F1 to stop recording
5. Set loop count (or 0 for infinite)
6. Press F2 to play back your recording
7. Press ESC or F2 to stop playback
